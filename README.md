## Installation ‚öíÔ∏è

Silakan git clone repository ini. Lalu ikuti langkah berikut

1. Buka terminal
2. Jalankan di terminal

```bash
composer install
```

3. Jalankan di terminal

```bash
php artisan migrate
```

4. Jalankan di terminal

```bash
npm run dev
```

5. Jalankan proyek

```bash
php artisan serve
```
